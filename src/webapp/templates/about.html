About me

Welcome to my personal cookbook! My name is Dennis Hilhorst and I set this project up so I can easily track, save and edit recipes that I like. I have kept expanding on it, and perhaps it is even useful to others! The recipes on the main page are recipes that I made together with my girlfriend Merel or friends. There is another "collection" of recipes that I have not made yet, but which looked good and which I intend to make at some point. You can find these through the buttons on the main page.

If you wish to contact me, please do so via the GitHub repository, or via chef(at)dennishilhorst.nl.
